使用流程

1

上传作答

拍照或扫描学生作答，连同参考答案一起上传。

2

AI 自动批改

系统识别文字与公式，对照评分标准逐步打分。

3

教师复核

对高风险样本进行人工复核，确认或改判得分。

4

查看结果

浏览班级看板、单份报告，获取讲评建议。

适用场景

面向教师

  • 批量批改作业和考试试卷，减少重复劳动
  • 通过复核工作台聚焦异常样本，提升批改质量
  • 利用错因统计和讲评建议，高效准备课堂讲解
  • 任务进度实时可见，随时了解批改状态

面向学校

  • 将教师经验性阅卷流程转化为可追溯的数字化工作流
  • 在保留教师最终判断权的前提下提升批改一致性
  • 沉淀题目级错误画像和复核记录，积累教学数据资产
  • 支持试点部署，渐进式推广，无需一次性替换现有流程
