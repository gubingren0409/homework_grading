教师侧 AI 阅卷与复核工作台

教师任务创建向导

把原来的“接口拼装页”改成老师能顺着业务流走的创建入口：先选批改模式，再确认评分标准来源，最后提交学生作答并跟踪进度。

当前页面已同时覆盖单学生多页、班级批量、参考答案与学生作答一站式提交三种主场景。

遗留批处理工具页 独立进度页（开发） 班级结果看板 任务列表 历史结果 报告展示页 复核工作台

适用工作流

  • 已有标准答案时，先生成评分标准，再批量提交学生作答。
  • 若老师已积累了 rubric，可直接选已有 rubric 快速开任务。
  • 若要一次性试跑新题，可用“一站式提交”直接上传参考答案和学生作答。
